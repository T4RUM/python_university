# 🐍 Aula 2 — Comentários em Python

## O que são comentários?

Comentários são partes do código que **não são executadas pelo Python**. Eles servem apenas para explicar o código para quem está lendo.

O interpretador Python ignora qualquer texto que venha após o (símbolo: #)


### O Python lê o código:

- da esquerda para a direita
- de cima para baixo

Quando encontra o símbolo `#`, tudo que estiver depois dele na mesma linha é ignorado.

```
# Isso é um comentário
print("Olá, mundo!")
```
---
## Para que usar comentários?

Comentários ajudam a:
* Explicar o que o código faz
* Facilitar manutenção futura
* Organizar o código
* Ajudar outras pessoas a entenderem o programa
---
## Conclusão

Comentários são essenciais para manter o código claro e organizado, principalmente em projetos maiores.
