# 🐍 Aula: Exercício — Variáveis e tipos de dados

## 🎯 Desafio do exercício
Neste exercício, o objetivo é praticar a criação de variáveis usando os tipos de dados que aprendemos até agora.

O desafio consiste em:

- Criar variáveis contendo informações de uma pessoa;
- Usar tipos diferentes de dados;
- Exibir os valores na tela usando `print()`.

⚠️ **Importante:**  
Se você estiver vendo este material para estudo, tente resolver o exercício **sem olhar o arquivo `.py` primeiro**.

A melhor forma de aprender programação é tentando resolver o problema antes de ver a solução.

---
## 🧠 O que precisamos criar?
Precisamos criar variáveis contendo:
- Nome
- Sobrenome
- Idade
- Ano de nascimento
- Se a pessoa é maior de idade
- Altura em metros

Cada informação utiliza um tipo diferente de dado.

---
## 🧱 Tipos utilizados no exercício
Durante o exercício usamos:

| Informação | Tipo |
|------------|------|
| Nome e sobrenome | `str` |
| Idade | `int` |
| Ano de nascimento | `int` |
| Maior de idade | `bool` |
| Altura | `float` |

## 🧠 O que aprendemos com este exercício?
Este exercício ajuda a entender:
- Como criar variáveis
- Como guardar diferentes tipos de dados
- Como usar comparações
- Como exibir dados formatados
- Como verificar tipos

Esse é o primeiro passo para criar programas mais complexos.

---
## ✅ Resumo do exercício
- Criamos variáveis para armazenar dados.
- Usamos tipos diferentes (str, int, float, bool).
- Calculamos se a pessoa é maior de idade.
- Exibimos valores e tipos na tela.
- Reforçamos o uso de variáveis na prática.
