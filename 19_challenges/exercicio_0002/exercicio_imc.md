# 🐍 Exercício: Cálculo do IMC + uso de Ellipsis

Nesta aula vamos praticar conceitos aprendidos até agora através de um exercício.

O objetivo é treinar:

- criação de variáveis
- uso de operações matemáticas
- uso do print formatado
- organização de dados em variáveis
- e conhecer um novo recurso do Python chamado **Ellipsis**.

### ⚠️ Importante: tente resolver sozinho antes de olhar a solução.

---
## 🧩 Desafio

Você deverá criar variáveis para armazenar:

- nome de uma pessoa
- altura
- peso

Depois disso, deverá calcular o **IMC (Índice de Massa Corporal)** usando esses valores.

Por fim, exiba uma frase mostrando:
- nome
- peso
- altura
- valor do IMC calculado

Use seus próprios dados ou valores fictícios para testar.

---
## 📐 Como calcular o IMC?
O cálculo do IMC é feito pela fórmula:
````python
IMC = peso / altura²
````
Onde:
- peso é dado em quilogramas
- altura é dada em metros
- altura² significa altura multiplicada por ela mesma
---
## ```…``` O que são esses três pontinhos no código?
Em alguns exemplos você verá:
```python
...
````
### Isso se chama Ellipsis.
Ele pode ser usado como um placeholder, ou seja, um marcador indicando que aquele trecho de código ainda será escrito.

Exemplo:
````python
imc = ...
````
Isso indica que o cálculo ainda precisa ser implementado.

O interessante é que o Python aceita isso sem gerar erro, permitindo deixar partes do código para completar depois.

Mais usos do Ellipsis aparecerão futuramente.
